# 山城夏影 - Galgame项目上下文

## 项目基本信息
- 游戏名称：山城夏影
- 引擎：Ren'Py 8.x

## 已完成剧情内容

### 第一章：漫展角落的黑玫瑰
1. 初遇：七月山城漫展场外，主角偷拍疲惫的安雅被发现
2. 核心分支选项（初遇被抓包时）：
   - 选项1：诚恳道歉，主动递上相机 → 主线剧情（完整30分钟）
   - 选项2：顺势搭话，想要加个好友 → 暧昧拌嘴分支（约20分钟）
   - 选项3：慌乱辩解，说"不就是拍张照吗" → 坏结局"尴尬收场"（约5分钟）
3. 主线关键节点：
   - 安雅看中主角的摄影能力，邀请他担任品牌摄影师和设计师
   - 两人在老城区寻找拍摄地，下雨躲进冰粉店
   - 安雅分享自己做独立品牌的梦想和过往经历
   - 主角送安雅回宿舍，安雅第一次露出温柔的一面
   - Demo结尾：第一次正式拍摄开始，留下悬念

## 全局变量

```renpy
default branch_type = 0       # 分支类型：0=主线 1=搭讪分支 2=坏结局
default anya_affection = 0    # 安雅好感度，影响后续对话和剧情
```

## 代码规范（必须严格遵守）
- 角色定义：主角用无名称角色 `define 我 = Character(None, kind=narrator)`，安雅用 `define anya = Character("安雅", color="#ff6b9d")`
- 场景切换：使用 `scene bg_xxx with fade` 格式，背景名统一用小写加下划线
- 角色表情：使用 `show anya xxx` 格式，如 `show anya pout`、`show anya smile`
- 心理描写：用第一人称括号格式 `我"（心理活动内容）"`
- 分支选项：使用 `menu:` 标签，每个选项对应一个独立的 label
- 音效：使用 `play sound xxx` 格式，如快门声 `play sound camera_shutter`
- 所有代码必须符合 Ren'Py 8.x 语法标准，可直接运行
